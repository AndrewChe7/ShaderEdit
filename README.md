# ShaderEdit

## Require
* node.js and npm
* nw.js
* fs
* three.js

## Uses
* nw.js
* three.js
* jquery
* materialize
* ace editor

## What is it?
It is simple fragmental and vertex shader editor. It uses ace as editor and three.js as renderer. Shaders must be written on GLSL. This project could be compiled for every OS with nw builder.
